This template is based on the Revealing Module Pattern.
The end goal is to simplify how you do organize your web apps or games.

#Notes#

*There is a manifest.json file to use if you want to publish your app to the
Chrome Store*

*There are words in directories like "components" and "scripts" it is just a
way to name your files. It's my personal preference, change as desired.*
